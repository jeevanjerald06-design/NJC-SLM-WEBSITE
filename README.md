## ‚úèÔ∏è How to Customize Your Website

### 1. **Edit Church Information**
Open `index.html` and find the comments marked with `<!-- EDIT THIS: -->`. Replace the placeholder text with your actual information:

- **Church History & Mission** (About section)
- **Founder Details** (Founder Name, Position, Background, Message)
- **Pastor Information** (Pastor Name, Position, Background, Message)
- **Contact Information** (Address, Phone, Email, Service Times)

### 2. **Add Your Photos**
1. Create a folder called `images` in your website folder (already created)
2. Add your church photos to this folder
3. In `index.html`, replace image file names with your actual photo names:
   - `church-interior.jpg` ‚Üí Your church interior photo
   - `worship-service.jpg` ‚Üí A worship service photo
   - `community-event.jpg` ‚Üí Community event photos
   - `choir-performance.jpg` ‚Üí Choir or special event photos

Example:
```html
<img src="images/your-photo-name.jpg" alt="Description">
```

### 3. **Add Your Videos**
For YouTube videos:
1. Go to your YouTube video
2. Click "Share" ‚Üí "Embed"
3. Copy the video ID from the embed code
4. Replace `dQw4w9WgXcQ` with your video ID in the `src` attribute

For other video platforms, use their embed codes.

### 4. **Customize Colors (Optional)**
If you want to change the color scheme, edit `style.css`:
- Primary Color: `--primary-color: #2c3e50;` (Dark blue)
- Secondary Color: `--secondary-color: #8b7355;` (Brown)
- Accent Color: `--accent-color: #d4af37;` (Gold)

### 5. **Change Website Title**
In `index.html`, the page title appears in browser tabs:
```html
<title>NEW JERUSALEM CHURCH SALEM BRANCH</title>
```

## üÜò Troubleshooting

**Images not showing?**
- Check file paths in image `src` attributes
- Make sure image files are in the `images` folder
- Use forward slashes in file paths: `images/photo.jpg`

**Videos not playing?**
- Use YouTube embed codes
- Check that video URL is correct
- Ensure video is public (not private)

**Website looks weird?**
- Clear browser cache (Ctrl + Shift + Delete)
- Try a different browser
- Check that all files (HTML, CSS, JS) are in the same folder
